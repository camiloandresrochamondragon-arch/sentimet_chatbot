🧠 Ingeniería de datos

Una vez comprendido el negocio, se exploran los datos disponibles. Esta sección combina análisis técnico con sensibilidad hacia lo que los datos representan.

📊 Actividades principales

  • Recolección de fuentes de datos internas y externas
  • Evaluación de calidad, completitud y relevancia
  • Tratamiento de datos faltantes, duplicados o inconsistentes
  • Transformación de variables para facilitar el análisis
  • Diseño de estructuras de datos

🔍 Preguntas guía

  • ¿Qué datos tenemos disponibles?
  • ¿Qué datos necesitamos recolectar?
  • ¿Qué tan confiables y actualizados son?

Esta fase prepara el terreno para el modelado, asegurando que los datos sean útiles, limpios y emocionalmente respetuosos con las personas que representan.
